# Homebrew Tap for MCP Vector Search

This is a [Homebrew](https://brew.sh/) tap for [mcp-vector-search](https://github.com/bobmatnyc/mcp-vector-search), a CLI-first semantic code search tool with MCP (Model Context Protocol) integration.

## Installation

```bash
brew tap bobmatnyc/mcp-vector-search
brew install mcp-vector-search
```

## Usage

After installation, you can use the `mcp-vector-search` command:

```bash
mcp-vector-search --version
mcp-vector-search --help
```

## Updating

To update to the latest version:

```bash
brew update
brew upgrade mcp-vector-search
```

## Uninstallation

```bash
brew uninstall mcp-vector-search
brew untap bobmatnyc/mcp-vector-search
```

## About

MCP Vector Search provides semantic code search capabilities with support for:
- Multiple programming languages
- Vector-based similarity search
- MCP server integration for AI assistants
- CLI-first design

For more information, visit the [main repository](https://github.com/bobmatnyc/mcp-vector-search).
